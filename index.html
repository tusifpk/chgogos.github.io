Θεωρία

Εργασία

Η εργασία θα πρέπει να υλοποιηθεί σε C++ ή Java ή Python. Αναλυτικές οδηγίες μπορείτε να διαβάσετε στην εκφώνηση της άσκησης.
Η εργασία είναι υποχρεωτική για όσους δηλώνουν το μάθημα για πρώτη φορά και έχει συμμετοχή 30% στην τελική βαθμολογία. Η εργασία είναι προαιρετική για τους υπόλοιπους σπουδαστές.
Η ημερομηνία παράδοσης είναι η 15-Δεκεμβρίου-2014. Παράδοση της εργασίας μετά τις 15-Δεκεμβρίου-2014 θα επιφέρει ποινή στη βαθμολόγηση κατά 40%. Σε κάθε περίπτωση η εργασία δεν μπορεί να παραδοθεί μετά τις 15-Ιανουαρίου-2015.

Μπορείτε να χρησιμοποιήσετε τον κώδικα που θα βρείτε στο github.com/chgogos/ett_ce_teiep_2014_2015 ο οποίος διαβάζει τα δεδομένα και τα τοποθετεί σε κατάλληλες δομές δεδομένων που επιτρέπουν τη μετέπειτα χρήση τους. Πρόκειται για ένα eclipse project το οποίο μπορείτε να κάνετε import στη δική σας εγκατάσταση. Στο ίδιο eclipse project θα βρείτε τα δεδομένα που απαιτεί η άσκηση (αρχεία μαθημάτων, σπουδαστών και λύσεων) καθώς και το αρχείο ett_help.pdf που περιέχει ένα σχεδιάγραμμα που επεξηγεί σε κάποιο βαθμό τις δομές δεδομένων που χρησιμοποιούνται στον κώδικα.

Εργαστήριο

Εργασία Α

Η πρώτη εργασία αφορά την παραγωγή τυχαίων αριθμών, την εγγραφή δεδομένων σε αρχεία, την ανάγνωση δεδομένων από αρχεία, αλγορίθμους ταξινόμησης και την χρονομέτρηση εκτέλεσης κώδικα. Μπορείτε να κατεβάσετε τον κώδικα που περιλαμβάνεται στην εκφώνηση από εδώ
Η λύση της πρώτης εργασίας

Εργασία Β

Η δεύτερη εργασία εξετάζει τον αλγόριθμο ranksort
Η λύση της δεύτερης εργασίας

Εργασία Γ

Η τρίτη εργασία εξετάζει τους αλγoρίθμους της σειριακής και της δυαδικής αναζήτησης
Η λύση της τρίτης εργασίας

Εργασία E
